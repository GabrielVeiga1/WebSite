Sobre o Site

O que Fazemos

Meu site foi criado com o objetivo de fornecer o conhecimento que eu aprendi durante meu curso de Dev Web. Aqui estão algumas das principais funcionalidades que o site oferece:

  • Gerenciamento de Cursos:
    • Listagem de Cursos: Visualize todos os cursos disponíveis com detalhes como nome, descrição e carga horária (CH).
    • Adicionar Novo Curso: Cadastre novos cursos facilmente através de um formulário simples e direto.
    • Atualização de Cursos: Edite as informações dos cursos existentes para mantê-los sempre atualizados.
    • Remoção de Cursos: Exclua cursos que não são mais necessários com apenas um clique.
  • Exploração de Filmes:
    • Filmes Populares: Descubra os filmes mais populares do momento.
    • Filmes para Crianças: Encontre filmes adequados para crianças.
    • Filmes por Ano: Explore filmes lançados em anos específicos, como 2010.
    • Gêneros Específicos: Navegue por filmes de gêneros específicos, como drama.
    • Atores Específicos: Veja filmes de atores famosos, como Tom Cruise.

Tecnologias Utilizadas

Nosso site é desenvolvido utilizando uma série de tecnologias modernas e robustas, garantindo uma experiência de usuário fluida e eficiente. Aqui estão algumas das principais tecnologias que utilizamos:

  • Flask: Um microframework para Python que nos permite criar aplicações web de forma simples e rápida.
  • SQLite: Um banco de dados leve e eficiente, perfeito para aplicações de pequeno a médio porte.
  • SQLAlchemy: Um ORM (Object Relational Mapper) para facilitar a manipulação do banco de dados.
  • Bootstrap: Um framework front-end que nos ajuda a criar interfaces responsivas e esteticamente agradáveis.
  • APIs Externas: Utilizamos APIs externas para fornecer dados dinâmicos e atualizados, como a API do The Movie Database (TMDb) para informações sobre filmes.

Como Utilizar

Navegue pelo menu superior para acessar as diferentes seções do site. Na seção de cursos, você pode adicionar, visualizar, atualizar e remover cursos. Na seção de filmes, explore diferentes categorias e descubra novos filmes para assistir. Na página inicial, adicione suas frutas favoritas e veja a lista crescer.

Estamos sempre trabalhando para melhorar nossa plataforma e adicionar novas funcionalidades. Esperamos que você aproveite ao máximo tudo o que oferecemos!
